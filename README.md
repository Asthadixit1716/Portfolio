# 🌐 Astha Dixit - Portfolio Website

This is my personal developer portfolio website built using **HTML**, **CSS**, and **JavaScript**, designed to showcase who I am as a MERN Stack Developer.

## ✨ Features

- 🎯 Clean and professional design
- 🖼️ Projects section with screenshots and descriptions
- 👩‍💻 About section with education and career overview
- 📄 Downloadable resume button
- 📬 Contact form connected to Telegram Bot for real-time messages
- 💡 Fully responsive and mobile-friendly

## 🚀 Technologies Used

- HTML5
- CSS3 (Flexbox, Responsive Design)
- JavaScript (DOM, Events)
- Telegram Bot API (for contact form)

## 📥 Resume

Download my resume directly from the site by clicking the "📄 Download Resume" button.

## 📬 Contact

The contact form sends your message straight to me via Telegram bot — fast and easy!
