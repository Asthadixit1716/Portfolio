Spatial Information System

Spatial Information System is a Web GIS application developed using ASP.NET Core MVC and PostgreSQL. It provides secure user authentication with registration, login, profile management, and password change. The application integrates GeoServer through WMS services to display geographic layers such as land use and vegetation on interactive maps.
